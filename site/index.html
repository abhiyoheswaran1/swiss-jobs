Each rectangle is an occupation — size is number of jobs, color is the selected metric. This is not a report or economic forecast — it is an exploratory tool for visualizing Swiss labour data.

Methodology & Data Sources

Employment: Eurostat lfsa_egai2d, 2023, Switzerland, ISCO-08 two-digit level, age 15-64. Wages: BFS Wage Structure Survey (LSE) 2022, median gross monthly wages by Berufsgruppe (ISCO-08). AI Exposure: Each occupation scored 0-10 by Claude (Anthropic) following Andrej Karpathy's methodology — work done entirely on computers faces the highest exposure. Scores do NOT predict job disappearance; they indicate the degree of AI-driven transformation.
